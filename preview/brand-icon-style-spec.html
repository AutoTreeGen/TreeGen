Iconography — 3D modern

The brand's icon language: each glyph is a chunky soft-body 3D object floating on a tinted backdrop, lit by a warm key from the upper left. No outlines, no medallion frame, no flat fills — shading carries the form.

Anatomy of one icon

  • Backdrop. A 1:1 rounded card (radius 18–24px) with a radial gradient: light tint at top-left → mid → deeper tone bottom-right. Six tints rotate per row (pink / cyan / gold / mint / plum / coral).
  • Body. One or more solid shapes filled with a multi-stop radial gradient (key 0.32, 0.22). Five stops: highlight → light → midtone → terminator → shadow. Never flat fill.
  • Specular. A small white ellipse at upper-left of each body, fill url(#b3Spec), ~30% of the body width.
  • Ambient bounce. A cool blue ellipse near bottom-right, url(#b3Ambient), mix-blend-mode screen, opacity 0.55. Optional but adds depth.
  • Hole / opening. Use the dark url(#b3Hole) radial (deep purple inset) for keyholes, doorframes, lens centers — never solid black.
  • Cast shadow. A blurred black ellipse on the ground plane (cy=85, unified across all 24 icons), opacity 0.16, blurred 4px.
  • Soft drop-shadow filter on the whole glyph group: filter:url(#b3Shadow) — adds a 6px contact shadow behind the object.

Anti-patterns

  • No medallion / coin container behind the icon. The glyph itself carries the brand.
  • No outlines or strokes on the body shapes (white interior strokes for chevrons / checks are fine).
  • No flat fills, no helix gradients, no chromatic palette on the glyph.
  • No emoji. No drop-shadow as emphasis.

Backdrop tint mapping

Each icon's backdrop tint is chosen by category — never random. New icons inherit tint by category; if ambiguous, pick the warmest tint that contrasts with the body color.

  • t1 pink — human / family / personal (person, book, place, magnifier-results).
  • t2 cyan — tools / search / data (magnifier, refresh, link).
  • t3 gold — temporal / records (clock, calendar, envelope, key, suitcase).
  • t4 mint — growth / tree / heritage (family tree, pedigree, house).
  • t5 plum — meta / system (gear, question, note).
  • t6 coral — affection / family bonds (heart).
